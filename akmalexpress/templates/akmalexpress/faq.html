{% trans "Вопросы и ответы" %}

{% trans "1. Как оформить заказ онлайн или в офисе?" %}

{% trans "Можно отправить ссылку менеджеру в Telegram/по телефону или прийти в офис. Мы считаем итоговую стоимость на месте, и если вас всё устраивает — сразу оформляем заказ." %}

{% trans "2. Какие сроки доставки?" %}

{% trans "AVIA занимает около 5-10 дней, Avto (Fura) — около 15-20 дней, iPost — около 20-25 дней." %}

{% trans "3. Как проходит оплата?" %}

{% trans "Для телефонов — полная оплата. Для обычных товаров, готовых к доставке, оплачивается себестоимость. Для станков и оборудования: 30-40% до подготовки к отправке из Китая, после подготовки — оставшаяся часть себестоимости, а по прибытию — карго и услуга." %}

{% trans "4. Какие способы оплаты принимаете?" %}

{% trans "Принимаем оплату картой, переводом по реквизитам (счет) и наличными в офисе." %}

{% trans "5. Можно ли изменить или отменить заказ?" %}

{% trans "Изменение или отмена возможны, если продавец еще не отправил товар." %}

{% trans "6. Что делать, если пришел не тот товар или есть брак?" %}

{% trans "До открытия упаковки обязательно снимите видео распаковки. По этому доказательству продавец может вернуть полную сумму, часть суммы или попросить возврат товара с последующим возвратом денег." %}

{% trans "7. Даете ли вы гарантию на товар?" %}

{% trans "Мы даем гарантию на доставку товара. Гарантию на сам товар предоставляет продавец." %}

{% trans "8. Как отслеживать заказ?" %}

{% trans "Можно любым удобным способом: по номеру квитанции, ФИО, номеру телефона или трек-номеру." %}

{% trans "9. Есть ли доставка по регионам Узбекистана?" %}

{% trans "Да, есть. По Узбекистану отправляем через BTS." %}
